Overview:
This project demonstrates professional Python development practices including modular programming, dependency management, and custom packaging. It transitions individual scripts into a reusable, production-ready structure.

Project Structure
- Modules: Contains refactored logic from Tasks 1, 2, and 3.
- Mypackage: A custom Python package containing age restriction and string utilities.
- Venv: Isolated virtual environment for project dependencies.
- Requirements.txt: List of required libraries for the project.

Setup Instructions - Create Virtual Environment: bash: python -m venv venv
