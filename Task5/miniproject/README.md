## Text File Analyzer

## Objective
This mini project demonstrates the application of Python fundamentals such as functions, object-oriented programming, file handling, error handling, and basic Streamlit integration.

## Problem Statement
Users often need quick insights from text files. This application allows users to upload a '.txt' file and get basic statistics such as word count, sentence count, and character count.

## Features
- Upload text files
- Count characters, words, and sentences
- Simple and clean UI using Streamlit
- Error handling for empty or invalid files

## Technologies Used
- Python
- Streamlit

## How to Run
-- bash
 ->  pip install -r requirements.txt
 ->  streamlit run app.py (or) python -m streamlit run app.py
